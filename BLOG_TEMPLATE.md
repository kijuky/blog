# Blog Template

以下は新規記事の基本テンプレート。

## 執筆ルール（固定）

- 冒頭に「結論3行」を置く（本題を先に示す）
- 1文1メッセージを意識して長文を分割する
- 「事実（観測）」と「考察（解釈）」を見出しで分ける
- 手順は `前提 / 手順 / 期待結果 / 実結果` をそろえる
- 最後に「今回の学び3つ」を置く

```markdown
# タイトル

## 結論（3行）

- 
- 
- 

## やろうとしたこと / 背景

- 

## 発生した課題

- 

## 解決方法

### 前提

- 

### 手順

1. 
2. 
3. 

### 期待結果

- 

### 実結果

- 

## なぜそれで解決できたか（理論）

## 事実（観測）

- 

## 考察（解釈）

- 

## 今回の学び3つ

1. 
2. 
3. 

## まとめ

- 
```
