例えば季節ものの掛け布団とかお客さん用の布団とか。我が家は押し入れがないため、こういった布団の収納ができず、部屋に放置されている。使っていない部屋があるので、現状はそれでも良いが、今後部屋を使いたくなったときに困るし、放置だけしていると湿気とか埃も気になってしまう。

そうして深夜にネットをザッピングしているとこんな商品を見かけた。

[rakuten:air-rhizome:10011407:detail]

見つけた瞬間、やられたと思った。普通、布団の収納はせいぜい収納棚とか圧縮袋みたいな商品しかないと思っていた。しかしこれは収納ではなくクッションとして利用しよう、というものだ。そこにあった布団がクッションとして別の機能を担ってくれる。同じ空間を占有しているはずなのに、収納だと空間が死んでるだけだけど、クッションだとそのクッションを利用することで空間が生きてくる。逆転の発想。これは収納革命だ。

さっそくポチってみたところ、想像通り良かった。敷布団を三つ折りしてクッションにしたてるのだが、三つ折りの布団は自立しないが、クッションカバーに入れることでこいつは自立する。使わない時は立てかけておけば空間を（それほど）圧迫しない。また、たまたま選んだサンゴマイヤーという生地だが、これが娘たちのお気に入りらしく、上機嫌になってくれるのも良い（娘たちは乳幼児のため、機嫌がよくなるアイテムは無条件で採用）。我が家は布団を上げ下げするのだが、布団を上げる代わりにこのクッションにしている。お客さん用の布団だけでなく、普段使いの布団をクッションにするのも便利だった。

ちなみに、似たようなものにぬいぐるみ収納ぬいぐるみみたいなのもあって、それもポチってみた。

[asin:B0DM6J66S2:detail]

ぬいぐるみとか使わなくなるとホコリ生成機になってしまうところを、使わなければクッションに入れるので、埃を纏りにくくなるし、この収納後のぬいぐるみ自体も可愛い。便利だと思って部屋中の（自分以外の）ぬいぐるみもこれに入れようと思ったら「ぬいぐるみの形が崩れそうで嫌」と言われた。まぁそれはあるかも。。

ぬいぐるみ以外にも、使わなくなったor着れなくなったけど（記念とかで）捨てられない捨てにくい服とかをこのぬいぐるみの中に入れて愛でるのも良さそうに思う。なかなか捨てられない服ってあるよね。そういうのをぬいぐるみとして再生できるのも良いかなと思った。他にも季節ものの服を入れておいて衣替えの時期に中身を入れ替えるとかしても良さそう。洗顔用タオルだけ入れておいてタオル置き場として使っても良さそう（中がタオルならクッション性も良さそう）。

# まとめ

今まで、使わない布製品はタンスの肥やしにしかならないと思っていた。それがクッションの中綿として利用することで、収納しつつも機能性を付加できる、スペパ（スペースパフォーマンス、スペース効率）の良いものに変えられることが分かった。こういうのは存在を知らないとたどり着くことがなかなか難しい（収納を増やすために「タンス」を検索しがち）ので、今回こういう商品を知らない人に刺さると良いなと思いました。
