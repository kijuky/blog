# 発生した課題

- WPSには「ボタン方式」と「PIN方式」の2種類があり、仕組みとリスクが異なる
- ボタン方式は物理アクセス必須だが、PIN方式は遠隔から総当たり攻撃可能で危険
- どの方式が有効になっているか分からず、安全性判断が難しい

# 課題の解決方法

- ルーター設定画面でWPS機能の種類（ボタン／PIN）を確認
- PIN方式がある場合は無効化
- ボタン方式のみの場合は、必要時だけ使用し、普段はOFFにする

# 解決方法の理論的説明

- WPSボタン方式は物理的操作が必要なため、遠隔攻撃が基本的に不可能
- WPS PIN方式は8桁の数字で接続が可能で、総当たり攻撃に弱く、遠隔からWi-Fi侵入されるリスクがある
- 無効化することで、総当たり攻撃や不正アクセスのリスクを排除できる
- ボタン方式のみの場合、必要なときだけ一時的にONにすることで利便性を確保しつつ、セキュリティを維持できる
