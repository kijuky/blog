# やりたかったこと

- 家庭内LANでIPv6を使用せず、全機器をIPv4で統一した運用にする。

https://mono96.jp/web/45596/

# 発生した課題

- NSDでIPv6ブリッジやRA配布が有効になっており、IPv6アドレスが割り当てられていた
- AXEにもIPv6設定が残っているため、ゲスト・メイン混在での挙動が不明

# 課題の解決方法

- NSDでIPv6ブリッジとRA配布を無効化
- AXEではIPv6機能をパススルーにして事実上使用不可に設定

# 解決方法の理論的説明

- IPv6 RAやDHCPv6を止めることで、LAN内の機器はIPv6アドレスを取得できなくなり、IPv6通信経路を完全遮断
- IPv4のみの運用にすることで、家庭内のネットワーク分離やセキュリティ管理が簡単になる
