# やりたかったこと

- ゲストSSIDやIoT機器を家庭内LANから隔離し、セキュリティを向上させる。

https://www.tp-link.com/jp/support/faq/1561/

# 発生した課題

- ゲストSSIDでも192.168.10.xの同一IP帯を割り当てるため、分離できていないのでは？と懸念
- mDNSやAirPrintが利用できない

# 課題の解決方法

- 実際にゲストSSIDでプリンタにpingやIP直打ちを試す
- ARPテーブル確認、MACアドレスを確認して通信遮断状況を観測

# 解決方法の理論的説明

- AXEのゲストネットワークはL2レベルでAP IsolationやARPフィルタを実装
- 同一IP帯でもブロードキャスト・マルチキャストは遮断されるため、LAN内通信は不可能
- これによりゲスト端末はインターネットのみアクセス可能、家庭内LANは完全隔離
