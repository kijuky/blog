各ディレクトリはホストしているブログサービス名です。

## 使い方

### service名/README.md

ブログサービスのURLを記載しています。

### service名/xx_ブログタイトル

ブログタイトルです。xxはASCIIでソートされます。

ブログタイトルに `/` が使われている場合、`_` に置換しています。

### service名/xx_ブログタイトル/README.md

ブログ本文です。公開されているブログの場合、先頭に公開URLを記載しています。

ブログ本文はmarkdown形式に書き直しています。GitHub上で大きく崩れない程度に修正しています。

### service名/xx_ブログタイトル/img

ブログ本文で使われている画像ファイルです。
