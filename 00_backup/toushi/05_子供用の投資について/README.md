2027年1月からこどもNISAが始まりますね。一般的に教育資金を貯めるために必要とされるサービスですが、そもそもお金がかかるのは0~2歳で、3~6歳は保育料無料だし、義務教育は授業料無料だったりします。教育資金はいつから貯めておくと良いのでしょうね？

先日、こども用の楽天銀行を開設できたので、そちらに資金をうつしつつ、楽天証券で全額をオルカン買っておきました。未成年口座なのでマネーブリッジができないですが、それでも楽天銀行と楽天証券の間の資金移動は楽でよいです。メインバンクが他にあっても楽天銀行を開設するのは意味あるでしょう。

こどもNISAが始まるまで1年間、銀行に預けっぱなしにするよりかは特定口座であってもオルカンを買っておいてインフレ負けしないようにします。こどもNISAが始まったら特定口座からNISA口座に移そうと思っています。こないだ、どこかのライオンが「特定口座からNISA口座に移さないほうが良い」みたいなようつべを見かけて、内容がくだらなかった（少なくとも私が気にするようなことではなかった）ので、大物ユーチューバーが言うこと全部間に受けない方が良いなぁとか思いました。

同じ某学長が、こどもNISAについても解説していて贈与税あたりのことを「行間読んでうまくやりや〜」的なことを言っていて、解説系動画なのになんでそこ解説しないんだろう？ってケチくさいなぁと思いました。ここで言うところの「行間」は、贈与税が発生する110万円/年を超えない程度に、毎年コツコツと子供用口座に積立する、NISA枠は60万円/年なので、残りは50万円/年で、それは特定口座につっこんどく、みたいなことかなと。

そういえば、18歳までこどもNISAなのだから、最大600万円(10年間)じゃなくて、1080万円(18年間)にすればいいのに、とか思いました。10歳になった時にまた考えるのめんどくさい。
