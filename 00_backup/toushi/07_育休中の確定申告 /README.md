年始は確定申告期間ですね。皆さん確定申告終わりましたか？私はついさっき終わりました。私は普通のサラリーマンで副業やFX等もしていない、口座はNISAと特定口座だけなので、確定申告はマイナカードをかざすだけで終わってしまいました。追加の郵送書類もない、イージーモードです。

私は現在、育休中なのですが、その場合、給与はゼロ、育休給付金は非課税扱いなので、低所得者扱いになります。そのため、確定申告では所得＜控除になり、源泉徴収額全額が返ってきました。非課税で月30~20万円もらいつつも、年末調整や確定申告で所得税分戻ってくるので、育休期間は意外とお金に困らなかったなという感じでした。

もしこれから育休を検討している人は、今の給与や雇用状況にもよりますが、取れるならがっつり取っちゃっても良いかもなぁとか思います。
