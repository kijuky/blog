乳幼児の三大支出といえば、服とミルクとオムツです。中でも乳幼児を持っている家庭に何か送るとしたらオムツが無難です。服は好みがあるだろうし（女児なら尚更）、ミルクは母乳育児とかを気にしている家庭だとめんどくさいです。その点、オムツは無難です。乳幼児はほっといてもおしっことうんこはするので、オムツはいくらあっても足りません。外部から供給が来ると尚助かります。最近だとオムツケーキという、オムツでできたケーキを送ることもあるらしいですね。

こちらのオムツ、何が良いかといえば、お尻を置く位置が分かる「おしりガイド」があることです。

ところで、オムツで困ることといえばなんでしょうか...？それはオムツからはみ出したおしっこやうんこです。ただでさえオムツの処理が必要なのに、そこからはみ出した汚物は服を汚してくるので、洗い物が増えるという厄介ごとが追加されます。これはストレスです。そしてはみ出す原因はオムツをきちんと履けていないことです。特に男性だと、自身でパンツに仕込む経験が乏しいため、おしりを置く位置を控えめにしがちです。その結果、汚物処理をする羽目になります。それがパートナーだった場合、「相手がオムツをきちんと履かせていなかったからこんな面倒ごとが増えてしまったんだ...！」みたいな疑心暗鬼を抱きながら汚物処理するわけです。これは精神衛生上良くない。

さて、そういう状況下で「おしりガイド」があるとどうなるでしょうか？おしりガイドのおかげでおしりの設置ミスがゼロになります。つまりパートナーに対して疑いの目を向けることがゼロになります。それでも汚物がはみ出すことはありますが、その場合は大体おむつのキャパオーバーが多いです。子供が原因の場合、それは成長の証として捉え直すこともできます。おしりガイドさまさまな訳です。
