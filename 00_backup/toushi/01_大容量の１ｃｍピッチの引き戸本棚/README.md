多分、ここの「商品レビュー」って中華商品とかを紹介するのだろうけど...w 吉田さんのチャンネルでもおうちの話あったし、家具ってそれなりな値段するから色んな意見あると参考になるかな、って。

というわけで、先日こんな家具を買いました。

[大容量の１ｃｍピッチの引き戸本棚（壁面収納）｜通販のベルメゾンネット](https://www.bellemaison.jp/shop/commodity/0000/1049385)

ソファ横のスペースが余っていたので、以前はオープンラックを置いていたのですが、中身が見えないように扉のある棚を選択。ソファ横に配置するため、扉は開戸ではなく引き戸に限定。するとベルメゾンのこの家具がヒットしました。

ベルメゾンなどの家具を買う場合、ベルメゾン本体に加えて、楽天支店からも同商品をチェックしておくと良いです。特にベルメゾンの場合は本体価格に送料が含まれているため、楽天によるポイント還元の恩恵を受けやすいです。また、こういったサイトでは最大限安くなっても10%引きなので、10%引きを上回るポイント還元が得られるなら楽天から買うと良いです。仮にサイトから買うとしても楽天リーベイツを経由して買うと良いです。

ベルメゾンの家具は自分で組み立てる方式なので、家具搬入はしやすいです。またこちらの家具は組み立ても大人一人で対応できます。ただし、素人が組み立てることを想定しているため、できあがった家具自体はちょっと安っぽい印象を受けます。背の高い引き戸に惹かれたのですが、実際に触ってみるとかなり軽くて不安定感があります。サイズが合うなら、[こちらの家具](https://www.dinos.co.jp/p/1300401186/)のような、中位の高さの引き戸の方が安定感があるように思います。

結果的には、ちょっと失敗したかなーという感じでした。もし、次に同じような家具を揃えようと思うなら、[こちらの家具](https://kaguyasujp.com/products/%E3%83%8A%E3%83%81%E3%83%A5%E3%83%A9%E3%83%AB-%E5%A4%A9%E7%84%B6%E6%9C%A8-%E5%BC%95%E3%81%8D%E6%88%B8-%E5%BC%95%E3%81%8D%E5%87%BA%E3%81%97-%E3%82%B5%E3%82%A4%E3%83%89%E5%8F%8E%E7%B4%8D-%E3%83%88%E3%83%83%E3%83%97%E5%8F%8E%E7%B4%8D-%E3%83%AF%E3%83%BC%E3%83%89%E3%83%AD%E3%83%BC%E3%83%96-%E3%82%AF%E3%83%AD%E3%83%BC%E3%82%BC%E3%83%83%E3%83%88-%E9%A0%91%E4%B8%88-%E9%AB%98%E5%93%81%E8%B3%AA-%E3%82%B7%E3%83%B3%E3%83%97%E3%83%AB-%E3%82%AB%E3%82%B9%E3%82%BF%E3%83%9E%E3%82%A4%E3%82%BA%E5%8F%AF%E8%83%BD-yg-k026?variant=44008755953861)の方が安いしシリーズで揃うのでいいかなと思いました。

誰かの家具選びの参考になれば幸いです。
