# この記事で分かること

Mac に AWS CLI をインストールできます

# ステップガイド

1. [python 3](https://www.python.org/downloads/) をインストールします
2. [pip](https://pip.pypa.io/en/stable/installing/) をインストールします
3. [aws cli](https://aws.amazon.com/jp/cli/) をインストールします
4. [プロファイル](http://docs.aws.amazon.com/ja_jp/cli/latest/userguide/cli-chap-getting-started.html)を追加します
